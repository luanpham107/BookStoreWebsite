# BookStoreWebsite
## Brief description:
This is a website for online books store.

Admin:
- Can create/edit/update/delete users/books/reviews.
- Can view order and sale report.

User:
- Can view Best sales, Statistics, Review a book.
- Add books into cart.
- Checkout.
- Payment with Paypal.

Technology:
- Servlet
- JSP
- Hibernate
- Javascript
- Jquery
